# AI Interview Agent
An AI-powered mock interview platform that generates technical questions, evaluates answers, and provides feedback in real-time.

---

## ✨ Features

* 🎯 Role-based question generation (Data Analyst, ML Engineer, etc.)
* 🧠 AI-powered answer evaluation with feedback
* 📊 Score-based performance tracking
* ⚡ Fast and interactive UI

---

## 🛠 Tech Stack

**Frontend**

* React (Vite)
* Modern UI components

**Backend**

* FastAPI
* LLM integration (Groq / OpenAI)

---

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/bhanudupati/AI-Interview-Agent.git
cd AI-Interview-Agent
```

### 2. Backend setup

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### 3. Frontend setup

```bash
cd frontend
npm install
npm run dev
```
